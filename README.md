# Telco Customer Churn Prediction -- End-to-End ML System

An end-to-end **Machine Learning project** for predicting customer churn
in the telecom domain, covering the full lifecycle from data
preprocessing and model training to **production-grade model serving
using FastAPI and Gradio**.

# Features

-   Data loading and validation
-   Robust preprocessing pipeline
-   Hyperparameter tuning with Optuna
-   Logistic Regression model
-   REST API using FastAPI
-   Web UI using Gradio


# Run Server

uvicorn app:app --reload
